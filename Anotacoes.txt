Var = Variáveis ---> 

-> number   
    Infinity
    NaN
-> string
-> boolean (true or false)
-> null
-> undefined
-> object
    Array 
-> function

Para saber o tipo de uma var, é só digitar o comando "typeof"


Regras para nomear uma variável:

- Podem começar com letra, $ ou _
- Não podem começar com números
- É possivel usar letras ou números
- É possível usar acentos e símbolos
- Não podem conter espaços
- Não podem ser palabras reservadas
- Maiúscula e Minúscula no js faz diferença
- Tente escplher nomes coerentes para as Variáveis

O "+" do java script server para CONCATENAÇÃO e tembém para ADIÇÃO

Formas de formatação de string:
'Eu estou aprendendo n'      <- Não faz interpolação
'Eu estou aprendendo' + n    <- Usa concatenação
`Eu estou aprendendo ${n}`   <- Usa template string


Number.parseInt(n)                Se quiser convertar para Inteiro
Number.parseFloat(n)              Se quiser convertar para Real
Number(n)                         Converte automaticamente para Inteiro ou Real

String(n)                         Converter para string 
n.toString()                      Converter "n" pra String

s.length               // quantos caracteres a string tem
s.toUpperCase()        // tudo para 'MAIÚSCULAS'
s.toLowerCase()        // tudo apra 'minúsculas'


Formatando Números

var n1 = 1543.5
n1 = '1543.5'

//se eu quiser q ela tenha 2 casas decimias
n1.toFixed(2)
'1543.50'

//se eu quiser mudar o "." para ","
n1.toFixed(2).replace('.', ',')
'1543,50'

//agora, se eu quiser colocar em moeda de qualquer país
n1.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'})
'R$ 1,543.50'
n1.toLocaleString('pt-BR', {style: 'currency', currency: 'USD'})
'US$ 1.543,50'
