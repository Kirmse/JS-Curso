No Js, o sinal de igual "=" sempre sera lido como RECEBE, nunca como igual, mas somento o sinal único "=".

Var = Variáveis ---> 

-> number   
    Infinity
    NaN
-> string
-> boolean (true or false)
-> null
-> undefined
-> object
    Array 
-> function

Para saber o tipo de uma var, é só digitar o comando "typeof"


Regras para nomear uma variável:

- Podem começar com letra, $ ou _
- Não podem começar com números
- É possivel usar letras ou números
- É possível usar acentos e símbolos
- Não podem conter espaços
- Não podem ser palabras reservadas
- Maiúscula e Minúscula no js faz diferença
- Tente escplher nomes coerentes para as Variáveis
